# dh-leyu-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文档的形式存放，不绑定任何特定域名或网站，也不构成完整的网站项目。

## 项目简介

该仓库旨在收集、整理并分享一些独立的 HTML 页面。每个页面通常包含特定的内容或功能，彼此之间没有强关联。适合作为个人笔记、小工具页面或临时演示文件的长期存档。

## 目录说明

仓库根目录下存放各 HTML 文件，部分页面可能附带独立的资源文件夹（如 CSS、JS 或图片）。目录结构大致如下：

```
/
├── index.html          （仓库入口或导航页，可选）
├── page-example.html   （示例页面）
├── another-page.html   （其他页面）
└── assets/             （资源文件夹，按需存在）
    └── style.css
```

每个 HTML 文件均为独立可用的页面，可直接在浏览器中打开查看。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容可能包含个人笔记、实验性代码、学习记录或临时工具。
- 部分页面可能包含外部链接或引用，但不保证长期有效。
- 归档内容会不定期更新或清理，不保证所有页面长期可用。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎通过 Issues 提出建议或报告问题（如页面显示异常）。
- 不接受未经沟通的 Pull Request 用于添加外部页面。
- 仓库内容可能随时调整，建议定期关注更新。

## 使用建议

- 直接下载或克隆仓库后，在本地用浏览器打开 HTML 文件即可。
- 也可通过 GitHub Pages 功能在线访问（需自行配置）。
- 页面中不包含任何跟踪代码或广告，可放心使用。
